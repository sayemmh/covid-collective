### How to use

- Fork the repository to your GitHub
- Edit [**profile.json**](https://github.com/vmarcosp/minimalist-profile/blob/master/src/profile.json) and insert your infos

`profile.json`

### Requirements

- Node v8+

### Development

```sh
yarn dev
# or
npm run dev
```

```sh
yarn build
# or
```
